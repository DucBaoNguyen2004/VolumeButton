# Volume Button Assistant

Ứng dụng nút Home ảo hỗ trợ điều chỉnh âm lượng nhanh chóng và tiện lợi.

## Tải xuống (Download)

Bạn có thể tải file cài đặt APK trực tiếp tại đây:

[![Download APK](https://img.shields.io/badge/Download-APK-blue?style=for-the-badge&logo=android)](https://github.com/DucBaoNguyen2004/VolumeButton/raw/main/app/release/app-release.apk)

> **Lưu ý:** Nếu link trên không tải được ngay, bạn có thể vào thư mục `app/release` và tải file `app-release.apk` thủ công.
